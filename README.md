# 🎮 Whack-A-Mole Game

An interactive and modern **Whack-A-Mole browser game** built using **HTML, CSS, and JavaScript**.
This project focuses on clean UI design, smooth animations, and engaging gameplay with multiple difficulty levels.

---

## 🚀 Features

* 🎯 **Level Selection** – Easy, Medium, Hard
* ⏱️ **Timer-Based Gameplay**
* 🐹 **Animated Mole with Pop-Up Effects**
* ✨ **Smooth UI Animations & Transitions**
* 🎴 **Game Over Card with Blur Effect**
* 🔙 **Navigation Controls (Back, Restart)**
* 📱 **Responsive Design (Mobile-Friendly)**

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling, Animations, UI Design
* **JavaScript (Vanilla JS)** – Game Logic & Interactivity

---

## 🎮 How to Play

1. Select a difficulty level (Easy / Medium / Hard)
2. Click on the mole 🐹 when it appears
3. Score points before time runs out ⏱️
4. Try to beat your best score 🎯
